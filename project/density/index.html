Density

Brief: Density by the Application Development Initiative (ADI) at Columbia University estimates how full study spaces are on campus using a live feed of Wi-Fi access data that updates every 15 minutes. Data is anonymized as device counts over a 15 minute period.

The application is built using Python and Javascript.

Execution: Following the public release of the first version of Density (complete with press coverage), I quickly threw together a hi-fidelity mockup with proposed beautifying and a new logo. I then coded the changes, which were quickly merged -- all within a couple days! My first open source contribution!

Tools: Adobe Illustrator, HTML/CSS
